KissLog Frontend
======================

KissLog.Frontend represents an user-interface application where users (developers, business analysts, QA testers, application managers) can visualise the captured logs, exceptions and other metrics data.

KissLog.Frontend displays the data by consuming KissLog.Backend REST services.

KissLog.Frontend can be :doc:`configured <configuration>` by updating the ``Configuration\KissLog.json`` file.
